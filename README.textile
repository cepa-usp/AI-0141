h1. Populações, comunidades e ecossistemas

Este Recurso Educacional (RE) visa promover a diferenciação entre os níveis de organização dos seres vivos: átomos, moléculas, organelas, moléculas, tecidos, órgãos, sistemas de órgãos, organismos, populações, comunidades e ecossistemas. Trata-se de um RE de interação simples do tipo _drag&drop_ (arrasta-e-solta).
